simpl.info Screen capture

This demo currently requires Chrome Canary with screen capture support enabled from about:flags.

View the console to see what's happening. The RTCPeerConnection objects pc1 and pc2 are in global scope, so you can inspect them in the console as well.

Code in this example used by kind permission of Vikas Marwaha and Justin Uberti.

For more information about RTCPeerConnection, see Getting Started With WebRTC.
