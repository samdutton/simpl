simpl.info ServiceWorker

This page uses ServiceWorker to cache page assets. To try this out, disable connectivity and reload the page (not hard reload).

Note that the page must be accessed using HTTPS.

View the browser console for logging from this page and navigate to chrome:serviceworker-internals for ServiceWorker logging (as implemented in /sw.js).
